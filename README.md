# Technical Accessment

1 - update the openai api key in  ```.env```

```bash
OPENAI_API_KEY=OPENAI_API_KEY
  
```


2 - Install ```node_modules```

```bash
yarn  
```

or

```bash
npm i
```

3 - Run the project
```bash
yarn dev
```

or

```bash
npm dev
```
